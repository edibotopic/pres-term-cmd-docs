## What are the ten ways?

The presentation broadly focuses on:

1. In-line commands
2. Code blocks
3. Screenshots
4. Terminal image generators
5. Auto-generated images
6. Screencasts
7. Video as text
8. Video as code
9. Interactive CLI
10. Bespoke code blocks
